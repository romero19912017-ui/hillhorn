## Описание

Проверка hillhorn_index_file и recall поиска.

## Задание для Agent

1. Индексируй README.md (или любой файл с описанием проекта)
2. Вызови hillhorn_search по ключевым словам из индексированного файла
3. Проверь, находит ли search индексированное

## Workflow

1. Прочитать SOUL.md или README проекта
2. hillhorn_index_file(file_path="SOUL.md", content=содержимое)
3. hillhorn_search("Agent identity NWF-JEPA")
4. Зафиксировать: нашел ли search
