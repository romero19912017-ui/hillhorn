# Задача 2: Теплый контекст (с памятью)

## Описание

Повторная задача после того, как в памяти уже есть данные (например, из task_01).
Проверяем, находит ли search прошлые решения.

## Задание для Agent

Найди в памяти информацию о greet.py (из task_01) и напиши функцию `bye(name)` в том же стиле.

## Workflow

1. hillhorn_get_context
2. hillhorn_search("greet.py утилита приветствия")
3. Если нашел - использовать стиль, создать bye.py
4. hillhorn_add_turn("bye.py - прощание, стиль как greet", kind="code")

## Чеклист проверки

- [ ] search вернул релевантное (про greet)
- [ ] bye.py создан в том же стиле
- [ ] add_turn вызван
